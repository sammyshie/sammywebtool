📐 知識整理三部曲（提示詞設計原則）

📥

🛠️ 知識再輸出工具鏈

📊 簡報後製工具：DeckEdit / PDNob / Canva▼
DeckEdit：免費線上 OCR 轉檔，將 NBLM 生成的簡報（PDF）轉成可編輯 PPTX，支援一鍵移除 AI 浮水印

PDNob PDF 編輯器：強大 OCR 後製神器，解決資訊圖表內文字亂碼問題，不讓排版跑掉

Canva Pro「魔法抓取」：將簡報背景與物件分離，方便重新排版設計
🧬 YAML 語法控制輸出風格▼
YAML 是一種結構化設定語法，可用來精確控制 NBLM 的簡報風格、資訊圖配色、字體大小等。讓每次輸出結果都一致！

範例：在 Prompt 前加上 YAML 框架，指定主色調、字體風格、版面結構，AI 就會按照你的規格生成。
🔗 Apps Script 自動化輸出 Google Sheets▼
利用 Google Apps Script 讓 NotebookLM 的 Data Tables 自動匯出至 Google Sheets，打造長期維護的關聯式資料庫，實現無縫自動化工作流！
